### 3. Strategic direction approval: Forecast of phase 2 
As mentioned in the statutes of PRAGMA :

Specific plans must be made in order to welcome applications from projects starting the fourth quarter of 2025. <br>
 - The Administrative Board takes on the Action Item to start Phase 2. 

### 4. Strategic direction approval: PRAGMA value proposition
PRAGMA as an organisation should provide the following value proposition:
- Open source mentorship: setting up and managing a group of willing mentors that will coordinate with projects that require their insights.
- Field tested documentation and workflows: consolidating in a single open source repository useful documentation for projects and breakdowns of workflows that can be helpful for open source projects.
- Assessment of maturity of an open source project: creating easy to use checklists to assess the state of a project and recommend a set of steps to improve the maturity level of a project
- PRAGMA ecosystem recommendation: building a set of existing open source solutions for specific use cases based on a set of criteria that will contribute to the project success

Everything mentioned above will be communicated in an open source manner. 
By unanimous vote the founding members of PRAGMA define that this will be a priority for the Administrative Board to work on and that this will be included in the quarterly meeting that the General Assembly members will be participating in. <br>
- The Administrative Board takes on the Action Item to start hosting quarterly meetings with the General Aseembly. <br>
- The Administrative Board takes on the Action Item to implement PRAGMA's Value Proposition.
